# All Downloader Pro

Professional All-in-One Downloader

## Features

- Download videos from YouTube
- Instagram
- TikTok
- Facebook
- X (Twitter)
- Reddit
- Threads
- Vimeo
- SoundCloud
- 1000+ supported websites via yt-dlp

## Run Locally

```bash
npm install
npm start
```

## Docker

```bash
docker build -t all-downloader .
docker run -p 8000:8000 all-downloader
```

## Deploy to Koyeb

1. Push this project to GitHub.
2. Create a new Koyeb Web Service.
3. Select the repository.
4. Koyeb will build the Dockerfile automatically.
5. The app will be available at your Koyeb URL.
